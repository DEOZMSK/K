# mr.Kcopoc_oplata_bot

## Калькулятор «🗓 Периоды»

- 1-й ряд формируется по фактическому дню недели из стандартной календарной функции Python (`date.weekday`), поэтому високосные годы учитываются автоматически.
- Для дат рождения 29 февраля действует единое правило: в невисокосные годы расчёт ведётся по 1 марта, в високосные — по реальному 29 февраля.
- 3-й ряд (основной цикл) автоматически пересчитывается на основе обновлённого дня недели, в то время как подпись годов (2-й ряд) и фоновое значение (4-й ряд) остаются неизменными по формуле.
